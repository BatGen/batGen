How do I use batGen?

It is highly recommended that you start out with an understanding of how to use MyBatis.

This product will not substitute the need for this knowledge.

For this tutorial, we will be using Eclipse with Maven installed.
  1. Download and extract files.
  2. Import batGen into Eclipse as existing Maven Projects.
  3. Create a new Maven project: select simple project, fill in required informations, click finish.
  4. Insert following dependencies into the pom.xml of the new project: 
    	
<dependencies>
    <dependency>
        <groupId>batgen.lib</groupId>
        <artifactId>batgen</artifactId>
        <version>1.0.0-RC1</version>
        <scope>test</scope>
    </dependency>
    <dependency>
        <groupId>junit</groupId>
        <artifactId>junit</artifactId>
        <version>4.8.1</version>
        <scope>test</scope>
    </dependency>
    <dependency>
        <groupId>org.mybatis</groupId>
        <artifactId>mybatis</artifactId>
        <version>3.1.1</version>
    </dependency>
    <dependency>
        <groupId>com.h2database</groupId>
        <artifactId>h2</artifactId>
        <version>1.3.171</version>
    </dependency>
</dependencies>
  5. Create a new file with the following example text and save as Employee.txt within src/test/resources:
      Symbols allowed in the .txt file, and their meaning.
    • /** documentation comment
    • * the field is required
    • - SequenceDisabled
    • ! the field is a primary key
    • ? the field is a search Id
    • // allow comments to be inserted after
    /**
* Employee table
**/
[Settings]
PACKAGE org.batgen.sample
CLASS       Employee
[Fields]
LONG(10)        employeeKey     EMPLOYEE_KEY!
DOUBLE(10,2)    salary       // Salary of employee
STRING(132)    lastName*      // Last Name
STRING(132)    firstName     // First Name
LONG(10)	supervisorKey	SUPERVISOR_KEY?
  6. Within src/test/java, create new Java Class Start and select public static void main(String[] args).
  7. Insert code into the Main of the newly created class. Import the required package org.batgen.CodeGen.
    • Takes 2 parameters: resource location and package name.
    CodeGen codeGen = new CodeGen( "src/test/resources", "org.batgen.sample" );
codeGen.run();
  8. Select Run As Java Application, and press Enter when prompted.
  9. Refresh project and new files should be created.
